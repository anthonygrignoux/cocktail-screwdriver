Tools :

["global" = doit être installé globalement, "save-dev" sera installé à l'exécution de npm install]

* Bower (global)
http://bower.io/
Installation des librairies JS

* Grunt (global + save-dev)
http://gruntjs.com/
Lanceur de tâches

* Zetzer (save-dev)
https://github.com/brainshave/grunt-zetzer
Pour compiler des pages html à partir de partials

* grunt-flats (save-dev)
https://github.com/adamduncan/grunt-flats
Pour compiler des pages html à partir de partials

* grunt-modernizr (save-dev)
https://github.com/Modernizr/grunt-modernizr
Pour générer automatiquement un fichier modernizr en fonction des tests utilsés dans le js si utile pour générer les classes)

* grunt-sass (save-dev)
https://github.com/sindresorhus/grunt-sass
Pour compiler sass sans passer par le compileur ruby compass 

* grunt-contrib-htmlmin (save-dev)
https://github.com/gruntjs/grunt-contrib-htmlmin
Pour minifier le html de distribution

* time-grunt (save-dev)
https://github.com/sindresorhus/time-grunt
Affiche le temps d'exécution des tâches grunt
